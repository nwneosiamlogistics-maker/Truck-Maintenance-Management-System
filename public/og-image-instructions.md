# OG Image Instructions

## Required: Create `/public/og-image.png`

For optimal social sharing (Facebook, Twitter, LinkedIn), create an OG image with these specifications:

### Dimensions
- **Size:** 1200 x 630 pixels (1.91:1 ratio)
- **Format:** PNG or JPG
- **Max file size:** 8MB (recommend < 500KB)

### Design Recommendations
1. **Include:**
   - Company logo (Neosiam Logistics)
   - App name: "Smart Truck Maintenance"
   - Tagline: "ระบบจัดการซ่อมบำรุงรถบรรทุกอัจฉริยะ"
   - Visual: Truck/fleet or dashboard screenshot

2. **Colors (from theme):**
   - Primary Blue: #1e40af
   - Accent Green: #10b981
   - Background: #0f172a (dark) or #f8fafc (light)

3. **Safe Zone:**
   - Keep text/logos within center 80% of image
   - Some platforms crop edges

### Quick Creation Options
1. **Canva:** Use "Social Media" → "Facebook Post" template
2. **Figma:** Create 1200x630 frame
3. **Screenshot:** Take dashboard screenshot and add overlay

### After Creation
1. Save as `og-image.png` in `/public/` folder
2. Test with: https://www.opengraph.xyz/
3. Clear social media cache after deploying
