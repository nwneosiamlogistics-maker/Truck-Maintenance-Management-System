# Truck Maintenance Management System

## Project
ระบบบริหารซ่อมบำรุงรถ, รายงานค่าใช้จ่ายรถ, อะไหล่มือสอง, ประวัติซ่อม, ประกัน/ภาษี

## Tech Stack
- React
- TypeScript
- TSX Components

## Rules
- ห้ามลบฟังก์ชันเดิมถ้าไม่จำเป็น
- แก้ทีละ feature หรือทีละ component
- ทุกครั้งก่อนแก้ ต้องอธิบายผลกระทบ
- รักษา UI เดิม ยกเว้นผู้ใช้สั่งให้ปรับ
- หลังแก้ให้ตรวจ TypeScript/build เสมอ

## Commands
- npm install
- npm run dev
- npm run build
